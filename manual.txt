To allow sudoers to execute command without password, follow below steps

Step 1 : sudo visudo

Step 2 : Add this entry in last line ->%superadmin ALL=(ALL:ALL) NOPASSWD:ALL

Step 3 : ctrl + x

Step 4 : y

Step 5 : Enter

Step 6 : sudo groupadd superadmin

Step 7 : sudo usermod -aG superadmin $USER



To add a scheduler to monitor the server health for every one minute, follow the below steps

Step 1 : crontab -e

Step 2 : Add this entry in last line -> */1 * * * * . /etc/profile; /usr/local/bin/servermonitor >> $MY_HOME/output.txt 2>&1

Step 3 : ctrl + x

Step 4 : y

Step 5 : Enter


Add this under .bashrc for accessing the log easily

alias alog="tail -n 500 -F $MY_HOME/output.txt"
alias applog="tail  -n 500 -F $MY_HOME/tomcat_build/logs/app_log.txt"
alias clog="tail  -n 500 -F $MY_HOME/tomcat_build/logs/catalina_log.txt"


Docker without sudo

sudo groupadd docker
sudo usermod -aG docker $USER
su - $USER



To generate self signed certificate

keytool -genkeypair -v  -keystore tomcat.jks   -keyalg RSA   -keysize 2048   -validity 365   -storepass tomcat   -keypass tomcat
